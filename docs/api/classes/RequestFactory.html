Class RequestFactory<T>

Provides a fluent way of configuring the RequestConfig and creating a {@link Request}. This class is used in the Client class and it's provided {@link Client.hooks.createRequest} hook.

Using the createRequestFactory function is the preferred way of creating an instance of this class as it returns the fluent RequestConfigSetter wrapper type that follows this class it's proxy behaviour.
