Optional Refresh

Refresh?: string
  • Defined in packages/@laravel-streams/api-client/src/types/headers.ts:746

Used in redirection, or when a new resource has been created. This refresh redirects after 5 seconds. Header extension introduced by Netscape and supported by most web browsers. Defined by HTML Standard

Optional Timing-Allow-Origin

  • Defined in packages/@laravel-streams/api-client/src/types/headers.ts:767

Optional X-Content-Type-Options

X-Content-Type-Options?: string
  • Defined in packages/@laravel-streams/api-client/src/types/headers.ts:781

The only defined value, "nosniff", prevents Internet Explorer from MIME-sniffing a response away from the declared content-type. This also applies to Google Chrome, when downloading extensions.

example

X-Content-Type-Options: nosniff

Optional X-UA-Compatible

  • Defined in packages/@laravel-streams/api-client/src/types/headers.ts:809
